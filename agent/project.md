### Especificação do Projeto

## 1. Contexto
Quero criar um sistema de eventos que permita o cadastro de usuários e eventos.

## 2. Objetivos
Nessa parte inicial preciso criar o frontend de um fluxo de cadastro de usuários. Para saber os campos necessários no cadastro, leia o arquivo database.puml e crie um fluxo de cadastro de usuários.

## 3. Escopo
Crie o frontend fluxo de cadastro de usuários.
Deve ser mobile first.

## 4. Definições técnicas
Framework: React / Next.js
Validação: Zod + React Hook Form
Linguagem: TypeScript
Estilização: TailwindCSS
Componentes: Shadcn UI
Quero login com redes sociais com Google e Facebook inicialmente.

## 5. Etapas de Implementação
Faça a configuração inicial do projeto e commite.
Só então comece as implementações de código.
